Collecte & Esquive
Description
"Collecte & Esquive" est un jeu dynamique où les joueurs doivent collecter des étoiles tout en esquivant des obstacles. Le jeu peut être joué seul ou avec un ami en mode coopératif ou compétitif.

Fonctionnalités
Modes de jeu solo, coopératif, et compétitif.
Collecte d'étoiles pour augmenter le score.
Évitement d'obstacles dynamiques qui augmentent en difficulté.
Support de contrôles pour deux joueurs: ZQSD pour le joueur 1 et les flèches du clavier pour le joueur 2.
Musique de fond et effets sonores pour améliorer l'expérience de jeu.

Prérequis
Naviguateur web moderne avec support de JavaScript.
Phaser 3

Installation
Ouvrez index.html avec live server pour lancer le jeu.

Commandes
Joueur 1 (rouge) : Utilisez les touches ZQSD pour vous déplacer.
Joueur 2 (bleu) : Utilisez les touches fléchées pour vous déplacer.
